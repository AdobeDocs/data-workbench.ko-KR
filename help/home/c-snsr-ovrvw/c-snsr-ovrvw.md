# 센서 개요{#sensor-overview}

센서는 웹, 응용 프로그램 및 데이터 수집 서버에서 안전하게 그리고 지속적으로 이벤트 데이터를 수집하여 데이터 워크벤치 서버로 전송하는 크기가 작고 효과적인 데이터 수집 소프트웨어 애플리케이션입니다.

[!DNL Sensor’s] 계기를 사용하면 인터넷 채널에서 발생하는 이벤트를 일관되게 정확하게 측정할 수 있습니다.
