# 하위 세트{#subsets}

데이터 하위 집합을 사용하면 데이터 세트에 있는 관심 요소만 선택하여 쿼리할 수 있습니다. 이렇게 하면 데이터 워크벤치 계산 속도가 빨라져 쿼리 결과가 빨라집니다.
