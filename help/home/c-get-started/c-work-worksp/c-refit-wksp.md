# 작업 영역 다시 맞춤{#refit-a-workspace}

{{eol}}

Refit Workspace는 시각화의 크기를 사용 중인 표시 해상도에 맞게 조정합니다.

작업 공간에서 **[!UICONTROL File]** > **[!UICONTROL Refit Workspace]**.

이 기능은 프레젠테이션의 표시를 빠르게 조정하는 데 유용합니다.
