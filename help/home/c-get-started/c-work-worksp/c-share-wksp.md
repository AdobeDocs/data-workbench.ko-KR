# 작업 영역 공유{#share-a-workspace}

작업 영역 내보내기 메뉴를 사용하여 작업 영역을 인쇄하고 .png 파일 또는 Excel로 내보낼 수 있습니다.

작업 영역에서 을 클릭합니다 **[!UICONTROL Export]**.
