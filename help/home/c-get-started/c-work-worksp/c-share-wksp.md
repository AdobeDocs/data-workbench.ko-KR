# 작업 영역 공유{#share-a-workspace}

{{eol}}

작업 공간 내보내기 메뉴를 사용하여 작업 공간을 인쇄 및 .png 파일 또는 Excel로 내보낼 수 있습니다.

작업 공간에서 **[!UICONTROL Export]**.
