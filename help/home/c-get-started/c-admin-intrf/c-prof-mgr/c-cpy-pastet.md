# 첨부 파일로 파일 복사 및 붙여넣기{#copy-and-paste-a-file-as-an-attachment}

Microsoft Outlook에서 파일을 전자 메일 첨부 파일로 복사하고 붙여넣는 단계입니다.

* 파일의 확인 표시를 마우스 오른쪽 단추로 클릭하고 을 클릭합니다 **[!UICONTROL Copy]**. 이메일 본문에 있는 Ctrl+v를 눌러 파일을 첨부합니다.
