# 작업 영역에 필터 적용{#apply-a-filter-to-a-workspace}

{{eol}}

작업 공간에 필터를 적용하는 절차.

* 필터 편집기 창에서 **[!UICONTROL Apply Filter]**.

   필터 편집기 창의 테두리가 노란색으로 바뀌어 필터가 작업 영역에 선택 사항을 적용했음을 나타냅니다.
