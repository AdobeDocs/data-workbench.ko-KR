# 프로세스 맵 이동 및 회전{#move-and-rotate-a-process-map}

{{eol}}

창 내에서 프로세스 맵을 이동하고 3D 프로세스 맵을 회전하는 방법에 대한 정보입니다.

**창 내에서 프로세스 맵을 이동하려면**

* 을(를) 클릭하고 맵을 원하는 위치로 드래그합니다.

**3D 프로세스 맵을 회전하려면**

* 마우스 오른쪽 단추를 클릭하고 원하는 회전 방향으로 마우스를 이동하면서 맵을 드래그합니다.
