# 노드 이동{#move-a-node}

맵이 동일한 기본 차원을 갖는 한 맵 내 또는 프로세스 맵 간에 노드를 이동할 수 있습니다.

예를 들어 3D 프로세스 맵에서 노드를 2D 프로세스 맵으로 드래그할 수 있습니다.

* 노드를 클릭하고 동일한 프로세스 맵이나 작업 공간의 다른 프로세스 맵에서 원하는 위치로 드래그합니다.
