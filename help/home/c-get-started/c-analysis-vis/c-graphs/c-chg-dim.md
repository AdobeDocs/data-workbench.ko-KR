# 차원 변경{#change-a-dimension}

차원을 변경하는 단계입니다.

* 변경할 요소 또는 차원의 레이블을 마우스 오른쪽 단추로 클릭하고 **[!UICONTROL Change Dimension]** > *&lt;**[!UICONTROL dimension name]**>*&#x200B;을 클릭합니다.
