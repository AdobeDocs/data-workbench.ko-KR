# 주석{#annotations}

{{eol}}

주석은 사용자가 작업 공간에 설명, 그래픽 또는 동적 제목을 포함하도록 추가할 수 있는 창입니다.

설명선으로 작업 공간에 주석을 추가할 수도 있습니다. 주석 콜아웃 작업에 대한 자세한 내용은 [작업 공간에 설명선 추가](../../../../home/c-get-started/c-vis/c-call-wkspc.md#concept-212b09e763044d938987b4a9c658adc0).
