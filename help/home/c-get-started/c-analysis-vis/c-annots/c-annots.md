# 주석{#annotations}

주석은 작업 영역에 설명, 그래픽 또는 동적 제목을 포함할 수 있는 창입니다.

설명선으로 작업 공간에 주석을 추가할 수도 있습니다. 주석 설명선 작업에 대한 자세한 내용은 작업 공간에 [설명선 추가를 참조하십시오](../../../../home/c-get-started/c-vis/c-call-wkspc.md#concept-212b09e763044d938987b4a9c658adc0).
