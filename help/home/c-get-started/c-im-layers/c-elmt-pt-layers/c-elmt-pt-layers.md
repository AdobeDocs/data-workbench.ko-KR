# 요소 점 레이어{#element-point-layers}

{{eol}}

Data Workbench에서 요소 점 레이어는 치수의 각 요소에 대해 전경에 한 점을 표시합니다.

전경에 표시되는 점의 크기는 요소를 플롯할 차원에 대해 지정된 지표를 평가하여 결정됩니다. 따라서 더 큰 지표 값은 전구에서 더 큰 포인트에 해당합니다.
