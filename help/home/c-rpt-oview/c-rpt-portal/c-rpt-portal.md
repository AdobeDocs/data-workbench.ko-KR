# 보고서 포털 사용{#using-the-report-portal}

보고서 포털을 사용하면 웹 브라우저를 통해 보고서 서버에서 생성된 보고서를 볼 수 있습니다.

이 [!DNL Report Portal] 응용 프로그램은 Microsoft IIS(인터넷 정보 서비스) 웹 서버에서 실행되는 웹 기반 응용 프로그램입니다.
