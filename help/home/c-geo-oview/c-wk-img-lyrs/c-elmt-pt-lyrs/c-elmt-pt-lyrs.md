# 요소 점 레이어 작업{#working-with-element-point-layers}

데이터 워크벤치에서 요소 점 레이어는 차원의 각 요소에 대해 지구 상의 한 지점을 표시합니다.

지구에 표시되는 점의 크기는 요소를 플롯할 차원에 대해 지정된 지표를 평가하여 결정됩니다. 따라서 지표 값이 클수록 지구 상의 더 큰 포인트에 해당합니다.
