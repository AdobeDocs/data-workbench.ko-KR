# 벡터 레이어를 사용한 작업{#working-with-vector-layers}

데이터 워크벤치에서 벡터 레이어는 정적인 벡터 데이터(라인 아트)를 전세계에 표시합니다.

벡터 레이어를 사용하여 주, 국가, 섬 또는 물의 본체와 같은 지리적 영역의 윤곽선을 표시할 수 있습니다.
