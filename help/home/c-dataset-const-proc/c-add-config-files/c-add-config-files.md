# 추가 구성 파일{#additional-configuration-files}

데이터 집합 프로필 또는 상속된 프로필의 데이터 집합 디렉터리에 들어 있는 일부 추가 구성 파일에 대한 정보입니다.

이러한 파일이 데이터 집합 프로필을 정의하는 데 반드시 필요한 것은 아니지만 데이터 처리를 위한 추가 매개 변수를 지정하도록 편집할 수 있습니다.
