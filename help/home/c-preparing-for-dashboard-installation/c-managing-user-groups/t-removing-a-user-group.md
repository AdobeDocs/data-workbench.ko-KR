# 사용자 그룹 제거{#removing-a-user-group}

{{eol}}

1. 을(를) 클릭합니다. **[!UICONTROL X]** 시스템에서 제거할 그룹의 행 끝에 있는 아이콘

   그룹 1을 삭제할 것인지 묻는 메시지가 표시됩니다. 클릭 **[!UICONTROL Yes]**.

   작업이 성공하면 그룹이 성공적으로 제거되었음을 나타내는 메시지가 표시됩니다.
