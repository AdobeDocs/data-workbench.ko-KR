# 사용자 그룹 편집{#editing-a-user-group}

{{eol}}

1. 행 끝에 있는 톱니바퀴 아이콘을 클릭하여 그룹을 편집합니다.

   ![](assets/edit_user_group.png)

1. 그룹 설정을 편집하고 **[!UICONTROL Save]** 변경 사항을 저장하려면 을 클릭합니다.

   작업이 성공하면 사용자 그룹이 성공적으로 저장되었음을 나타내는 메시지가 표시됩니다.
