# 사용자 계정 제거{#removing-a-user-account}

{{eol}}

1. 을(를) 클릭합니다. **[!UICONTROL X]** 시스템에서 제거할 사용자의 행 끝에 있는 아이콘을 클릭합니다.

   사용자를 삭제할 것인지 묻는 메시지가 표시됩니다. 1. 선택 **[!UICONTROL Yes]** 를 눌러 사용자를 삭제합니다.

   작업이 성공하면 사용자가 성공적으로 제거되었음을 나타내는 메시지가 표시됩니다.
