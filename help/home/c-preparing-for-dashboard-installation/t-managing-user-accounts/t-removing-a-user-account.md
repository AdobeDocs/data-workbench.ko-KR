# 사용자 계정 제거{#removing-a-user-account}

1. 시스템에서 제거할 사용자에 대한 행 끝에 있는 **[!UICONTROL X]** 아이콘을 클릭합니다.

   사용자를 삭제할지 여부를 묻는 메시지가 표시됩니다. 1.사용자를 **[!UICONTROL Yes]** 삭제하려면 선택합니다.

   작업이 성공하면 사용자가 성공적으로 제거되었음을 나타내는 메시지가 표시됩니다.
