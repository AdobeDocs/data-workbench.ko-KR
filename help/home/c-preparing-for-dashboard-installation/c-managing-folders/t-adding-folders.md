# 폴더 추가{#adding-folders}

1. 기존 폴더를 마우스 오른쪽 단추로 클릭하고 **[!UICONTROL New subfolder]**&#x200B;선택합니다.

   ![](assets/new_subfolder_1.png)

1. 새 하위 폴더가 생성됩니다.

   ![](assets/new_subfolder_2.png)

   작업이 성공하면 폴더가 생성되었음을 나타내는 메시지가 표시됩니다.
