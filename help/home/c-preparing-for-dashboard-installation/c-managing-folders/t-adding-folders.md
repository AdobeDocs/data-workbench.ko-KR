# 폴더 추가{#adding-folders}

{{eol}}

1. 기존 폴더를 마우스 오른쪽 단추로 클릭하고 **[!UICONTROL New subfolder]**.

   ![](assets/new_subfolder_1.png)

1. 새 하위 폴더가 만들어집니다.

   ![](assets/new_subfolder_2.png)

   작업이 성공적으로 수행되면 폴더가 만들어졌음을 나타내는 프롬프트가 표시됩니다.
