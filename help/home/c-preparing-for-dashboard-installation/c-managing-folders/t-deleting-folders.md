# 폴더 삭제{#deleting-folders}

1. 삭제할 폴더를 마우스 오른쪽 단추로 클릭하고 **[!UICONTROL Delete]**&#x200B;선택합니다.

   ![](assets/delete_folder.png)

   >[!NOTE]
   >
   >폴더를 삭제하려면 먼저 비어 있어야 합니다.

1. 폴더를 삭제할 것인지 묻는 메시지가 표시됩니다.
1. 작업이 성공하면 폴더가 삭제되었음을 알리는 메시지가 표시됩니다.
