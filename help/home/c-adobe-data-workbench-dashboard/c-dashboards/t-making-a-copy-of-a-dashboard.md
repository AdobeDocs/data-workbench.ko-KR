# 대시보드 사본 만들기{#making-a-copy-of-a-dashboard}

대시보드를 복사하는 단계입니다.

1. 복사본을 만들 대시보드를 로드합니다.
1. 대시보드 인터페이스 **[!UICONTROL Save As…]** 상단의 대시보드 메뉴에서 선택합니다.
1. 창이 **[!UICONTROL Save Dashboard As…]** 나타나면 대시보드 사본의 새 이름을 입력합니다.
