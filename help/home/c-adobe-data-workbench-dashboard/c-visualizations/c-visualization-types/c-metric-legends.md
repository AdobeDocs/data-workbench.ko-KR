# 지표 범례{#metric-legends}

지표 범례 시각화를 사용하면 지표 합계를 표 형식으로 표시할 수 있습니다. 첫 번째 열에는 지표 이름이 포함되고 두 번째 열에는 지표 값이 포함됩니다. 모든 행을 한 번에 창에 표시할 수 없는 경우 스크롤 막대가 표시됩니다.

특정 차원 값이 표시되지 않으므로 지표 범례 내에서 선택할 수 없습니다.
