# 일반 시스템 관리{#general-systems-administration}

특정 Adobe 소프트웨어 제품 관리에 대한 자세한 내용은 이 안내서의 개별 장을 참조하십시오.

>[!NOTE]
>
>이 안내서, 권장 빈도 및 지침은 모니터링 도구를 사용하고 있다고 가정합니다.
